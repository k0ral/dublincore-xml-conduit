# dublincore-xml-conduit

This [Haskell][hsk] library implements an XML streaming parser/renderer for the **Dublin Core Metadata Element Set** Version 1.1 ([RFC 5013][dc]), based on [conduit][cdt]s.


[dc]: https://tools.ietf.org/html/rfc5013
[cdt]: https://hackage.haskell.org/package/conduit
[hsk]: https://haskell.org
